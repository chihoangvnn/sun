 1. Thư Viện Nội Dung (Content Library)
Content Types với Tags:
// Media content với tag system
contentLibrary = {
  id: varchar().primaryKey(),
  title: text(),
  content: text(),                    // Text content
  mediaUrls: string[],               // Video/hình ảnh URLs
  tagIds: string[],                  // Reference to unified_tags
  contentType: "text" | "video" | "image" | "mixed",
  platforms: string[],               // Auto from tags
  priority: "high" | "normal" | "low",
  usageCount: integer(),
  lastUsed: timestamp(),
  isActive: boolean(),
  createdAt: timestamp()
}
Tag-based Organization:
📁 #sản-phẩm (45 bài)
├── 📝 Text posts (20)

└── 🎥 Videos (10)
📁 #khuyến-mãi (30 bài)
├── 📝 Sales posts (20)
└── 🎥 Promo videos (10)
⏰ 2. Lịch Đăng Bài Thông Minh (Smart Scheduler)
Priority + Random System:
schedulingRules = {
  priorityPosts: string[],           // Manually selected priority content
  randomPool: string[],              // Auto-selected from library by tags
  timeSlots: {
    morning: "9:00-11:00",
    afternoon: "14:00-16:00", 
    evening: "19:00-21:00"
  },
  distribution: {
    accountId: string,
    postsPerDay: number,
    platforms: string[],
    timePreferences: string[]
  }
}
Multi-Account Distribution:
Account A (Facebook + Instagram):
├── 9:00 AM - Priority post (#sản-phẩm)
├── 2:30 PM - Random post (#nội-dung) 
└── 8:15 PM - Random post (#khuyến-mãi)
Account B (TikTok + Twitter):
├── 10:45 AM - Random post (#video)
└── 7:30 PM - Priority post (#event)
🔄 3. Đồng Bộ Đa Nền Tảng (Multi-Platform Sync)
Cross-Platform Publishing:
crossPlatformPost = {
  contentId: string,
  platforms: {
    facebook: { accountId, customizations },
    instagram: { accountId, customizations },
    tiktok: { accountId, customizations }
  },
  scheduledTime: Date,
  status: "pending" | "posting" | "completed"
}
